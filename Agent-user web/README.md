# Agent User Web v1.8.21

User-facing sandbox web app for Jarvis Agent Platform.

## Highlights

- Chat header and composer glass transparency / blur intensity are swapped.
- Profile Personal section includes a sandbox password-change reserve.
- Agent card voice preview uses a custom SVG three-arc sound icon instead of emoji text.
- Agent card button text is uppercase `START` with stronger hover-lift polish.

## Run

```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```

Open `http://127.0.0.1:5174`.
